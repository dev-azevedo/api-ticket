# API Tickets

## ‼ Rquirements

- [Uv python](https://docs.astral.sh/uv/getting-started/installation/)

## 🚦 How to start

```bash
git clone https://github.com/jhonatan-azevedo/api-tickets.git
```

```bash
cd api-tickets
```

```bash
uv sync
```

## 🏃🏼‍♂️ How to run

```bash
uv run fastapi dev src/main.py
```
